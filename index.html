Programação Orientada a Objetos

A disciplina de Análise de Sistemas II tem como principal objetivo proporcionar aos estudantes os fundamentos do paradigma orientado a objetos e modelagem de sistemas orientados a objetos. Para isto será utilizado a linguagem de programação Java.

Técnicas de Programação

Capacitar o aluno a analisar problemas e projetar, implementar e validar soluções para os mesmos, através do uso de metodologias, técnicas e ferramentas de programação que envolvam os elementos básicos da construção de algoritmos e programas de computador. Como objetivos específicos temos a compreensão de linguagens de programação estruturada (linguagem C) e melhoramento nas técnicas de resolução de problemas e raciocínio lógico.
